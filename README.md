## 📌 Overview 

This repository demonstrates the implementation of multiple machine learning classification models on three different datasets:

	•	🌸 Iris Dataset (flower classification)
 
	•	☀️ PlayTennis Dataset (weather-based decision making)
 
	•	🍷 Wine Dataset (wine quality classification)
 
The project compares different algorithms, evaluates their accuracy, and showcases how hybrid/ensemble approaches can outperform single models.

## ⚙️ Features
	•	✅ Preprocessing pipeline (missing values, normalization, encoding)
	•	✅ Feature selection & dimensionality reduction (PCA / statistical methods)
	•	✅ Hybrid classification approach (e.g., Logistic Regression + Random Forest + SVM)
	•	✅ Model stacking and ensembling for improved predictions
	•	✅ Performance evaluation with accuracy, precision, recall, F1-score, and ROC-AUC


## 🧠 Model Used
	•	Logistic Regression
	•	Decision Tree
	•	Random Forest
	•	K-Nearest Neighbors (KNN)
	•	Support Vector Machine (SVM)
	•	Naive Bayes
 	•	Random Forest
